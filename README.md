# ZSharpAsync
ZSharpAsync 是 ZSharp 的扩展插件，实现了一个适用于虚幻引擎的单线程异步模型。插件以**事件循环**和**定时器**为基础，集成了 .NET 的 **async/await** 和 **AsyncEnumerable**，旨在提高如网络通信、资源管理、Gameplay 等异步系统的开发效率。

**注意：本项目处于早期开发阶段，未经充分验证且未来会频繁出现无预警的破坏性改动，请不要用于发布产品。**
